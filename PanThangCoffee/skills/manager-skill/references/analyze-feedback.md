# การวิเคราะห์ความคิดเห็นลูกค้า (Analyze Feedback)

คำแนะนำนี้ใช้สำหรับดึงและวิเคราะห์ข้อมูลความคิดเห็นลูกค้า (Customer Feedback) ของร้าน Panthang Coffee

## ขั้นตอนการทำงาน
1. **ดึงข้อมูล:** ห้ามอ่านไฟล์ `data/feedback.csv` ตรงๆ ให้รันสคริปต์ `script/get_feedback.py` โดยระบุ `caller_role="manager"` เสมอ
2. **ข้อควรระวัง:** ห้ามสร้างข้อมูลหรือคำวิจารณ์ที่ไม่มีอยู่ในไฟล์ขึ้นมาเองโดยเด็ดขาด (No Hallucination)
3. **การจัดกลุ่มข้อมูล:** 
   - จัดกลุ่มเสียงตอบรับเชิงบวก (จุดที่ลูกค้าพึงพอใจ)
   - จัดกลุ่มเสียงตอบรับเชิงลบหรือข้อเสนอแนะ (จุดที่ลูกค้าต้องการให้ปรับปรุง)

## รูปแบบการแสดงผล (Output)
สรุปผลลัพธ์ภายใต้หัวข้อ **### [Customer Trend]**
- สรุปจำนวนข้อความฟีดแบคทั้งหมด และคะแนนเฉลี่ย
- สรุปประเด็นหลักๆ ที่ลูกค้าชื่นชม (จุดแข็งของร้าน)
- สรุปประเด็นหลักๆ ที่ลูกค้าบ่นหรือเสนอแนะ (จุดอ่อน/โอกาสในการพัฒนา)
- ห้ามนำ JSON ดิบจากสคริปต์มาแสดงตรงๆ ให้สรุปเป็นภาษาพูดของพี่ปั้น
