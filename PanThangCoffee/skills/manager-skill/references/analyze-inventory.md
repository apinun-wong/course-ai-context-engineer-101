# การวิเคราะห์สต็อกสินค้า (Analyze Inventory)

คำแนะนำนี้ใช้สำหรับดึงและวิเคราะห์ข้อมูลสต็อกสินค้า (Inventory) ของร้าน Panthang Coffee

## ขั้นตอนการทำงาน
1. **ดึงข้อมูล:** ห้ามอ่านไฟล์ `data/inventory.csv` ตรงๆ ให้รันสคริปต์ `script/get_inventory.py` โดยระบุ `caller_role="manager"` เสมอ
2. **เงื่อนไขเวลา:** ให้สมมติว่าวันที่ปัจจุบันคือ "2026-06-20" ใช้เป็นฐานในการคำนวณวันหมดอายุของสินค้า
3. **การคัดกรอง Critical:** สินค้าที่เหลืออายุ 3 วันหรือน้อยกว่า ต้องถูกตีตราว่า "⚠️ เสี่ยงหมดอายุ (Critical)"

## รูปแบบการแสดงผล (Output)
สรุปผลลัพธ์ภายใต้หัวข้อ **### [Inventory Status]**
- สรุปจำนวนรายการสินค้าทั้งหมด
- แจกแจงรายการสินค้าที่ใกล้หมดอายุ (เข้าเงื่อนไข Critical) เพื่อให้เจ้าของร้านรีบพิจารณาจัดการ
- สรุปรายการสินค้าอื่นๆ ที่ยังมีอายุเหลือเฟือในภาพรวม
- ห้ามนำ JSON ดิบจากสคริปต์มาแสดงตรงๆ ให้สรุปเป็นภาษาพูดของพี่ปั้น
