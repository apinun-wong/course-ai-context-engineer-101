# Contextual Engineer Prompt 101

คอร์สเรียนรู้การเขียน Prompt อย่างมีบริบท (Context) เพื่อสื่อสารกับ AI ได้อย่างมีประสิทธิภาพ

## สิ่งที่จะได้เรียนรู้

- เข้าใจหลักการของ Context Engineering และความแตกต่างจาก Prompt Engineering แบบเดิม
- เทคนิคการให้บริบทที่ชัดเจนและครบถ้วนแก่ AI
- การออกแบบ Prompt ที่ได้ผลลัพธ์ตรงตามต้องการ
- การประยุกต์ใช้งานจริงในสถานการณ์ต่าง ๆ

## เนื้อหาหลัก

### 1. Context Engineering คืออะไร

การออกแบบและจัดการบริบท (Context) ที่ส่งให้ AI อย่างเป็นระบบ ไม่ใช่แค่เขียนคำสั่งสั้น ๆ แต่คือการประกอบข้อมูลที่จำเป็นทั้งหมดเข้าด้วยกัน เพื่อให้ AI เข้าใจเจตนาและส่งมอบผลลัพธ์ที่ถูกต้อง

### 2. องค์ประกอบของ Context ที่ดี

- **Role** — กำหนดบทบาทให้ AI
- **Goal** — ระบุเป้าหมายที่ชัดเจน
- **Constraints** — ข้อจำกัดและเงื่อนไข
- **Examples** — ตัวอย่าง Input/Output ที่คาดหวัง
- **Format** — รูปแบบผลลัพธ์ที่ต้องการ

### 3. เทคนิคสำคัญ

- การใช้ System Prompt อย่างมีประสิทธิภาพ
- Few-shot Prompting ด้วยตัวอย่างที่เลือกมาอย่างดี
- Chain-of-Thought สำหรับงานที่ต้องใช้เหตุผล
- การแบ่งงานใหญ่ออกเป็นขั้นตอนย่อย

### 4. ฝึกปฏิบัติ

ใช้โปรเจกต์ PanThangCoffee เป็นกรณีศึกษาสำหรับฝึกเขียน Prompt ในสถานการณ์จริง
